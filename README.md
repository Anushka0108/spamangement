<h1>💆‍♀️ Spa Management Software</h1>
A desktop-based Spa Management System developed using C# (.NET) in Visual Studio with database integration. This application helps manage daily spa operations including appointments, customers, staff, and billing efficiently.<br>

<h2>📌 Features</h2>
👤 Customer Registration & Management<br>
📅 Appointment Booking System<br>
🗄️ Database Integration (SQL Server)<br>

<h2>🛠️ Technologies Used</h2>
C#<br>
.NET Framework (Windows Forms)<br>
SQL Server<br>
Visual Studio<br>

<h2>🎯 Project Objective</h2>
The goal of this project is to digitalize spa operations by replacing manual record-keeping with an efficient desktop application that improves accuracy, saves time, and enhances overall management.

<h2>🚀 How to Run the Project</h2>
Clone the repository<br>
git clone https://github.com/your-username/spa-management-software.git<br>
Open the solution file (.sln) in Visual Studio<br>
Configure the SQL Server database connection<br>
Run the project<br>
