<h1>💆‍♀️ Spa Management Software</h1>
A desktop-based Spa Management System developed using C# (.NET) in Visual Studio with database integration. This application helps manage daily spa operations including appointments, customers, staff, and billing efficiently.

📌 Features
👤 Customer Registration & Management
📅 Appointment Booking System
🗄️ Database Integration (SQL Server)

🛠️ Technologies Used
C#
.NET Framework (Windows Forms)
SQL Server
Visual Studio

🎯 Project Objective
The goal of this project is to digitalize spa operations by replacing manual record-keeping with an efficient desktop application that improves accuracy, saves time, and enhances overall management.

🚀 How to Run the Project
Clone the repository
git clone https://github.com/your-username/spa-management-software.git
Open the solution file (.sln) in Visual Studio
Configure the SQL Server database connection
Run the project
